# Drinking-BlackJack
_an AdHoc creation_

Welcome to _**Drinking Blackjack**_, a fun twist on classic blackjack. It includes custom drinking game rules designed to add extra motivation and excitement to the traditional game of BlackJack.

Unless stated otherwise, the normal rules of BlackJack are followed. However, here are some notable exceptions:
- Special rules regarding splitting aces
- _ALWAYS_ split 10s

The extended drinking rules can be found in `Rules.md`. These rules aren't set in stone, the best ones often come mid-game! Players are encouraged to come up with new rule ideas as they play. If they make the game more fun, they are probably worth keeping.

This repository will, at a later date, also include `BlackJack.py` to facilitate remote play of this game with friends.

## Drink Responsibly ⚠️
This game is best enjoyed in good company and with good judgment. **Drink responsibly and know your limits**.

The goal is to have fun, not regrets.

## Running the Game

Start the game by running:
```bash
python BlackJack.py
```

Alternatively, the rules of this game can easily be applied to a physical deck of cards if you want to play offline with freinds.
